# Code of Conduct

Hi! I'm the one person behind MCPower. Here's the whole code of conduct:

- **Be nice.** To me, to other users, to yourself. Pet a cat if one is nearby.
- **Don't be angry if I go quiet.** I answer when I can, but life is sometimes
  harsh and I'm not always around. A nudge after a while is fine; a pile-on is not.
- **Be patient with the code.** I've been building this for over two years and
  I genuinely forget what some parts do. If something looks weird, ask — it might
  be a bug, or it might be load-bearing weirdness I'd forgotten about.
- **Help keep it readable.** A lot of my (and Claude Code's) effort went into
  describing the code well. If you touch it, leave it at least as well-explained
  as you found it.

That's it. Don't harass people, don't be a jerk, and we'll get along great.
If someone's making this place unpleasant, email me at pawellenartowicz@europe.com
and I'll sort it out.
